iOS Viewport Unit Bug Test

  1. You should see a green strip at the very bottom of the screen, just above the browser's bottom toolbar. If you don't see it, your browser has the bug.
  2. Now scroll down slowly until the browser toolbar collapses.
  3. This blue section is sized with 100svh. Per the CSS spec, it must not resize when the toolbar hides. If it visibly jumps or grows, your browser has the bug.
  4. Scroll back to the top. The blue section and green strip should look exactly the same as when you started.

Per CSS Values Level 4, svh and lvh are stable values that do not change when dynamic browser UI (toolbars) expand or collapse. Only dvh should change. The green strip's height is calc(100lvh - 100svh), which equals the toolbar height and must be greater than zero.
