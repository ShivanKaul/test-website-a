Viewport Keyboard Resize Tests

Tests whether the browser correctly handles the interactive-widget viewport meta value when the virtual keyboard appears.

Default (no interactive-widget) Spec default is resizes-visual. Layout viewport should not change. interactive-widget=resizes-visual Explicit resizes-visual. Layout viewport should not change. interactive-widget=resizes-content Opt-in to resize. Layout viewport should shrink. interactive-widget=overlays-content Keyboard overlays everything. Neither viewport should change.
