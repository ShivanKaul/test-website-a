Storage Quota Fingerprinting Test

This test checks whether storage quota APIs return values that differ between normal and private browsing modes. If they do, a website can detect incognito mode.

How to use: Run this page in a normal window, note the values, then run it in a private/incognito window and compare. Identical values = good (no fingerprint). Different values = the API leaks browsing mode.

Related: brave-browser#39520
