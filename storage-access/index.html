Storage Access API Test Harness Site A

This page embeds Site B in an iframe and displays the embedded test results.

What the test checks

  1. It asks you to open the seed page on Site B. That top level page writes a known token into localStorage.saa_fp_token and cookie saa_fp_cookie on Site B.
  2. It loads Site B inside an iframe and checks whether that first party state is visible before any request.
  3. In the iframe, you click a button to call document.requestStorageAccess().
  4. It checks again and reports one primary status message plus a JSON summary.

Actions

Open Site B seed page
After you seed Site B, reload the embedded test if needed.
